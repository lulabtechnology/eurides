# Subir a GitHub desde la WEB (sin CLI) — PASO A PASO

## 1) Crear el repositorio
1. Entra a GitHub e inicia sesión
2. Arriba a la derecha, presiona el botón **+**
3. Elige **New repository**
4. Nombre del repo: `eurides-landing` (o el que prefieras)
5. Marca **Public** o **Private** (lo que necesites)
6. NO marques “Add a README” (ya hay uno en el proyecto)
7. Presiona **Create repository**

## 2) Subir archivos (Upload files)
1. En la página del repo recién creado, busca el botón **Add file**
2. Selecciona **Upload files**
3. Abre en tu computadora la carpeta del proyecto descomprimido
4. Selecciona TODO (carpetas y archivos) y arrástralos al área de upload
   - Debes subir: `app/`, `components/`, `lib/`, `public/`, y archivos raíz como `package.json`, etc.
5. Espera que termine de cargar

## 3) Confirmar el commit
1. Abajo verás “Commit changes”
2. En el mensaje, pon:
   - `Initial commit: Eurides landing`
3. Presiona **Commit changes**

## 4) Verificación rápida
1. Ya dentro del repo, revisa que existan estas carpetas:
   - `app`
   - `components`
   - `public/images`
2. Abre `package.json` y confirma que se ve el contenido.
